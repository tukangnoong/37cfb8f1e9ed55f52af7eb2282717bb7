2. Klik Entri baru, lihat gambar!

3. Lihat gambar di bawah ini dan baca keterangan/tutorial di bawahnya!

*keterangan gambar:

  • 1 : Isi dengan Nama Produk, hindari pengunaan karakter yang tidak perlu agar lebih SEO Friendly!
  • 2 : Pastikan anda berada di mode HTML!
  • 3 : Silahkan paste/tempel kode yang telah anda salin tadi dan edit/isi sesuai petunjuk!
  • 4 : Isi dengan kategori produk (contoh: Sepatu Pria), disarankan gunakan satu kategori saja!
  • 5 : Jika semuanya sudah benar, silahkan klik Publiskan untuk mempubliskan, Simpan untuk menyimpannya terlebih dahulu!

4. Selesai

Home Slideshow

Menambah konten Home Slideshow

1. Salin kode di bawah ini!

<b:widget id='ImageXXX' locked='true' title='Slide Baru' type='Image' version='1'>
<b:widget-settings>
<b:widget-setting name='displayUrl'>http://3.bp.blogspot.com/-Z2AbnceuNlo/XfhpMITwROI/AAAAAAAAG0g/Uw5iop5SpaU0vDwopXbEWYdgnA-6ctgbwCK4BGAYYCw/s1600/s3%2B%25281%2529.png</b:widget-setting>
<b:widget-setting name='displayHeight'>650</b:widget-setting>
<b:widget-setting name='sectionWidth'>150</b:widget-setting>
<b:widget-setting name='shrinkToFit'>false</b:widget-setting>
<b:widget-setting name='displayWidth'>1600</b:widget-setting>
<b:widget-setting name='link'><![CDATA[/search/label/Label%206?&max-results=8]]></b:widget-setting>
<b:widget-setting name='caption'>Keterangan Slide Baru</b:widget-setting>
</b:widget-settings>
<b:includable id='main'>
<b:tag class='lazy-bg slideIMG' cond='data:link' expr:data-src='data:sourceUrl' expr:href='data:link' expr:id='data:widget.instanceId + &quot;_img&quot;' expr:title='data:title + &quot; - &quot; + data:caption' itemprop='url' name='a'><span class='hide' itemprop='name'><data:caption/></span></b:tag>
<b:include name='quickedit'/>
</b:includable>
</b:widget>

2. Buka Notepad dan paste/tempel kode di atas!

3. Kembali ke Blogger, klik Tema > Edit HTML, lihat gambar!

4. Lihat gambar di bawah ini dan baca keterangan/tutorial di bawahnya!

*keterangan gambar:

  • 1 : Klik kolom editor, bebas, dimana aja!
  • 2 : tekan CTRL+F dan cari kelola home slideshow
  • 3 : Perhatikan yang saya lingkari kuning, dalam contoh ini sudah terdapat 3 konten Home Slideshow dengan id Image201, Image202 dan Image203, maka jika ingin menambahkan konten Home Slideshow baru, tentu saja harus menggunakan id Image204, caranya, silahkan kembali ke notepad dan edit kode yang tadi dipaste/ditempel, ganti ImageXXX dengan Image204!
  • 4 : Jika kode di notepad sudah diedit, silahkan salin semua dan paste/tempel di baris yang saya beri garis hijau!
  • 5 : Klik Simpan tema

4. Selesai! tinggal mengeditnya di beranda blog anda menggunakan tool yang sudah disediakan agar lebih mudah, tutorialnya sudah saya sertakan di bawah!

Menghapus konten Home Slideshow

1. Klik Tema > Edit HTML, lihat gambar!

2. Lihat gambar di bawah ini dan baca keterangan/tutorial di bawahnya!

*keterangan gambar:

  • 1 : Klik kolom editor, bebas, dimana aja!
  • 2 : tekan CTRL+F dan cari kelola home slideshow
  • 3 : Perhatikan yang saya lingkari kuning, dalam contoh ini terdapat 3 konten Home Slideshow dengan id Image201, Image202 dan Image203, untuk menghapusnya tinggal hapus aja baris kode dari konten yang akan dihapus!
  • 4 : Klik Simpan tema

4. Selesai!

Mengubah konten Home Slideshow

1. Buka beranda/halaman awal blog anda

2. Sorot konten Home Slideshow yang ingin diubah dan klik tombol edit

3. Setelah itu akan muncul jendela baru, lihat gambar dan keterangannya di bawah ini!

*keterangan gambar:

  • 1 : Judul
  • 2 : Keterangan
  • 3 : Hubungan atau Link tujuan ketika konten Home Slideshow ini diklik
  • 4 : Gambar Konten Home Slideshow, ukuran gambar 1920x780, klik hapus terlebih dahulu untuk menggantinya
  • 5 : Jika sudah selesai silahkan klik tombol Simpan

4. Selesai!
